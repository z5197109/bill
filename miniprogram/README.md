# 账单助手 - 微信小程序版

基于原有 React/Ant Design Web 应用转换而来的微信小程序版本。

## 项目结构

```
miniprogram/
├── app.js                 # 小程序入口
├── app.json               # 全局配置
├── app.wxss               # 全局样式
├── project.config.json    # 项目配置
├── utils/
│   ├── api.js             # API 请求封装
│   └── util.js            # 通用工具函数
├── pages/
│   ├── dashboard/         # 财务看板
│   ├── upload/            # 上传识别
│   ├── analytics/         # 消费分析
│   └── settings/          # 设置页面
└── assets/
    └── icons/             # 图标资源
```

## 功能模块

### 1. 财务看板 (Dashboard)
- 本月支出概览
- 预算状态显示
- 分类消费分布

### 2. 上传识别 (Upload)
- 选择/拍摄账单图片
- OCR 自动识别
- 编辑识别结果
- 批量保存

### 3. 消费分析 (Analytics)
- 日期范围筛选
- 分类筛选
- 关键词搜索
- 账单明细列表
- 统计数据展示

### 4. 设置 (Settings)
- 账本管理（新建、切换、删除）
- 分类管理（添加、删除）
- 规则管理（关键词自动分类）

## 使用方法

### 1. 配置后端地址

在 `app.js` 中修改 `baseUrl`：

```javascript
globalData: {
  baseUrl: 'http://your-server-address:5000',
  // ...
}
```

### 2. 导入微信开发者工具

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择 `miniprogram` 目录
4. 填入 AppID（测试可使用测试号）
5. 点击「导入」

### 3. 开发调试

- 在开发者工具中开启「不校验合法域名」选项（详情 → 本地设置）
- 确保后端服务已启动
- 编译预览

## 注意事项

1. **域名配置**：正式发布需要在小程序管理后台配置合法域名
2. **HTTPS**：生产环境需要配置 HTTPS
3. **图标资源**：TabBar 图标需要替换为 PNG 格式（建议尺寸 81x81）
4. **OCR 功能**：依赖后端 OCR 服务正常运行

## 技术栈

- 微信小程序原生开发
- 后端：Flask API（复用原有后端）
